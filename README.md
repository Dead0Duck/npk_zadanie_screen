# NPK Показывалка заданий

Мини-приложение на электроне (что его делает уже не мини, но не суть) для показа текста заданий в колледже.\
Требуется второй монитор, так как выводит на него.\

Стек: Electron + Vite + TypeScript + React + TailwindCSS

## Команды
`npm run dev` - Запустить версию для разработки\
`npm run build` - Скомпилировать приложение
